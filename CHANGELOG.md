# Changelog

All notable changes to the **Syniq (Simon Nexus)** project are documented in this file.

---

## [1.0.0] - 2026-07-21

### Added

- **Progressive Web App (PWA)**: Support offline capabilities, auto-updating service worker, and vector SVG web manifest configurations.
- **Testing Architecture**: Vitest + React Testing Library integration with 100% pass rates across game core validation steps, storage systems, and settings repositories.
- **Accessibility Enhancements**: Support color-blind filters, geometric shape labels (▲, ●, ■, ◆), screen reader details, and OS reduced motion overrides.
- **Advanced Game Modes**: Integrate Classic (Simon), Speed Rush, Reverse Mode, and Time Attack challenge variations.
- **App Settings Dashboard**: Volume sliders, animation playback controls, theme togglers, and IndexedDB local preferences persistence.
- **Vercel Deployments Config**: Integrated `vercel.json` SPA path rewrites.
- **SEO Configurations**: Configured meta descriptions, keywords, Open Graph metadata, and Twitter card tags.

---

## [0.5.0] - 2026-07-18

### Added

- **UI overhaul**: Sleek dark-themed glassmorphism cards and layout spring animations using Framer Motion.
- **Audio Synthesizer Service**: Programmatic chime generation via browser Web Audio API oscillator nodes.
- **Telemetry Storage**: Custom IndexedDB repository wrapper for player statistics and top 10 scoring logs.
- **Layout Routing**: React Router pages setup and lazy route transitions.
